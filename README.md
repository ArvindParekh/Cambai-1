## 🛠️ Installation

1. Clone the repository:

   ```bash
   git clone git@github.com:ArvindParekh/Cambai-1.git
   ```

2. Change into project directory:

    ```bash
    cd Cambai-1
    ```

3. Install dependencies:

    ```bash
    pnpm install
    ```

4. Start the application:

    ```bash
    pnpm run dev
    ```
